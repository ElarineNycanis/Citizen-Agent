# CitizenAgent 安装与使用指南

## 这到底是什么？

CitizenAgent 是一个运行在**命令行**里的 AI 助手。你可以用它：

1 **聊天** — 日常对话、问问题、写文案（DeepSeek 驱动）
2 **看图** — 把图片路径发给它，它能描述图片内容（Qwen 视觉模型）
3 **生图** — 让它画图，自动调用 AI 生成并保存到桌面（通义万相）
4 **跑命令** — 让它帮你查天气、下载文件、批量重命名（真实 shell）
5 **同时做好几件事** — 一句"帮我搜 A、画 B、查 C"，三个代理同时干活

> **它是终端程序，不是图形 App。** 如果你从来没打开过"命令提示符"，先往下看——其实很简单。

---

## 三步上手

### 第一步：安装 Python

1. 打开 https://www.python.org/downloads/
2. 点黄色大按钮下载
3. 运行下载的文件，**务必勾选底部 `Add Python to PATH`**，然后点 Install
4. 装完后关掉窗口

### 第二步：运行安装脚本

1. 双击 `安装指导\install.bat`
2. 等它跑完（大概 30 秒）
3. 如果提示"安装完成"，继续下一步

### 第三步：填 API Key

1. 打开 `安装指导\API_KEY_SETUP.md`，照着步骤获取 Key
2. 按 `Win + R`，输入 `%USERPROFILE%\CitizenAgentConfig.json`，回车
3. 在打开的记事本里，把 `api_key` 后面的占位文字替换成你的真实 Key
4. 保存，关掉

> 如果第二步出错，看下方的「手动安装」。

## 启动

双击项目目录里的 `CitizenAgent.bat`，会看到一个黑窗口：

```
==================================================
Citizen Agent
==================================================
[?] 是否信任该工具及当前环境？(y/n, 默认 n):
```

输入 `y` 回车。看到类似 `已连接 api.deepseek.com` 就说明成功了。

然后就可以直接打字聊天了：

```
> 你好，介绍一下你自己

林清墨: 你好！我是林清墨，你的 AI 终端助手...
```

## 日常用法举例

### 聊天
```
> 帮我用通俗的话解释一下什么是区块链
> 给我推荐三个周末自驾游的地方，从杭州出发
> 用幽默风格写一段介绍我家猫的朋友圈文案
```

### 看图（把图片路径发过去就行）
```
> C:\Users\admin\Desktop\我的猫.jpg 这是什么品种？
```
程序会自动调用视觉模型分析图片，然后回答你。

### 生图
```
> 帮我画一只穿宇航服的柴犬，背景是火星
```
AI 会调用 Wanx 生成图片，保存到桌面。也可以指定路径：
```
/create_picture 一只弹吉他的猫 C:\Users\admin\Desktop\cat.png
```

### 让 AI 帮你操作电脑
```
> 帮我把桌面所有 .png 文件移动到一个叫 screenshots 的文件夹里
```

### 同时做多件事
```
> 帮我搜索今天杭州的天气、查一下比特币价格、然后给我讲个笑话
```
三个代理并行干活，省时间。

## 常用命令

在聊天框里输入 `/` 开头的是特殊命令：

| 命令 | 作用 |
|------|------|
| `/help` | 查看所有命令 |
| `/history` | 查看、切换历史对话 |
| `/new` | 开始新对话 |
| `/delete` | 删除不需要的对话 |
| `/create_picture 描述` | 生图并保存 |
| `/quit` | 保存对话并退出 |

## 手动安装

如果 `install.bat` 跑不了，手动来：

```bash
# 1. 确认 Python 已装
python --version

# 2. 安装依赖
pip install requests
pip install prompt_toolkit    # 可选，让输入更好看

# 3. 创建配置文件
copy config.example.json "%USERPROFILE%\CitizenAgentConfig.json"
```

## 常见问题

**Q: 双击 bat 文件闪退？**
A: 在 bat 文件所在目录的地址栏输入 `cmd` 回车，然后输入 `Python CitizenAgent.py`，看报什么错。

**Q: 提示 `python 不是内部命令`？**
A: Python 没装好，或者安装时没勾选 `Add Python to PATH`。重装 Python 并确保勾选。

**Q: 提示 `模型 xxx 未配置`？**
A: 检查 `C:\Users\你的用户名\CitizenAgentConfig.json` 里的 `api_key` 填了没。

**Q: 聊天时提示 401 或鉴权失败？**
A: API Key 填错了，或者 Key 被删除了。去对应平台重新生成一个。

**Q: 回复很慢？**
A: 正常，API 调用需要几秒钟。看到转圈说明在等待回复。

**Q: 能在手机上用吗？**
A: 暂不支持。不过如果你用 Termux（Android 终端模拟器），理论上能跑。

**Q: 怎么备份我的对话？**
A: 所有对话存在 `conversations\` 文件夹里，复制这个文件夹就行。跨电脑可以用 `/history` 切换。
